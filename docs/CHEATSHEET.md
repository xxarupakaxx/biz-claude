# Claude Code チートシート

**印刷して手元に置ける早見表**

---

## 基本コマンド

| コマンド | 説明 |
|---------|------|
| `/guide` | ガイドツアー（初めての方） |
| `/skills` | 利用可能なスキル一覧 |
| `/lfg <タスク>` | 自律モードでタスク実行 |
| `/clear` | 会話をリセット |

---

## よく使うスキル（自然言語でOK）

### ドキュメント作成

| やりたいこと | 言い方の例 |
|-------------|-----------|
| 議事録作成 | 「議事録作成して」「MTGメモをまとめて」 |
| レポート作成 | 「レポートにして」「報告書を作って」 |
| 引き継ぎ資料 | 「引き継ぎ資料を作って」 |
| PRD作成 | 「PRDを書いて」「要件定義して」 |

### 調査・分析

| やりたいこと | 言い方の例 |
|-------------|-----------|
| リサーチ | 「〇〇について調べて」「競合を分析して」 |
| アイデア整理 | 「アイデアを整理して」「ブレストしよう」 |

### 知見管理

| やりたいこと | 言い方の例 |
|-------------|-----------|
| 記憶させる | 「これを覚えて」「保存して」 |
| 思い出す | 「〇〇について何を話した？」 |
| 知見を整理 | 「メモリを整理して」 |

### 改善・学習

| やりたいこと | 言い方の例 |
|-------------|-----------|
| 業務改善 | 「もっと効率化したい」「改善したい」 |
| 学習を実践 | 「この記事の内容を実践したい」 |
| 設定改善 | 「間違えたから設定を直して」 |

---

## 困ったときは

| 状況 | 対処法 |
|------|-------|
| 何ができるか分からない | 「何ができる？」と聞く |
| 前の会話を忘れた | 「〇〇について何を話した？」 |
| 指示が複雑 | `/lfg` でまかせる |
| コマンドを忘れた | このチートシートを見る |

---

## Tips

- **自然に話しかけてOK**: コマンドを覚えなくても大丈夫
- **曖昧でも大丈夫**: Claudeが質問してくれる
- **「覚えて」が便利**: 重要なことは覚えさせておく
- **`/lfg`が最強**: 複雑なタスクはまかせる
